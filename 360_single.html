VR for Oculus

Built in Unity for the Oculus Quest 2; this project uses 360 degree video to build an immersive environment for users to explore in virtual reality. Users are able to engage with world space UI elemets that help them navigate the digital environment, and learn about their surroundings at the same time. 360 video contributes to a fully animated environment in 3D.
